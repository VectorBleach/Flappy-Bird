Git Notes

  • create a repo ==> mkdir _name
  • change the working directory to _name ==> cd _name (repo created)
  • create a new file and list it to current repo ==> ls _nameFile
  • stage the file ==> git add _nameFile
  • check the status ==> git status
  • commit the file (add a message) ==> git commit -m "message.."
  • Create Branch (allow to work on diff parts of proj without impacting main branch) ==> git branch _branchName
  • Swith to newly created branch ==> git checkout _branchName
