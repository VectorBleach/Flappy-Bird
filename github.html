Github Notes

  • adding a remote repo with URL as an origin to your local git repo ==> git remote add origin https://YourUrl.git
  • push master branch to the origin url and set it as default remote branch ==> git push --set-upstream origin master
  • pull is combination of e diff commans 1) fetch 2) merge
  • fetch updates to see what has changed on github ==>git fetch origin
  • merge combines the current brancg with specified branch ==> git merge origin/master
